# Smarter BlinkIt

AI powered smart marketplace that connects buyers with local sellers.

## Features

- AI Shopping Assistant
- Intent Based Search
- Barcode Inventory System
- Smart Cart Splitting
- Product Graph Recommendations

## Tech Stack

Frontend: React
Backend: Node.js / Express
Database: PostgreSQL + Neo4j
AI: HuggingFace Models
Computer Vision: OpenCV
Payments: Razorpay

## Project Stages

Stage 1 – Foundation
Stage 2 – AI Automator
Stage 3 – Logistics Orchestrator
Bonus – Prediction Features
